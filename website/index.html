Zor Code

Open-source AI coding agent for the terminal

MIT License 27 Providers Bun TypeScript

Zor Code is an AI-powered coding assistant that lives in your terminal. Part of the Zor ecosystem — Code (terminal), Cowork (IDE), Chat (web). Multi-provider, local models, MCP support, sub-agents. Live model switching mid-chat across 27 LLM providers.

Install

# macOS / Linux
curl -fsSL https://raw.githubusercontent.com/zor-ai/zor/main/install.sh | sh

# Windows PowerShell
irm https://raw.githubusercontent.com/zor-ai/zor/main/install.ps1 | iex

# Homebrew
brew install zor-code

# npm
npm install -g zor-code

Quick Start

zor-code keys set opencode <your-api-key>
zor-code

Features

27 Providers
Anthropic, OpenAI, Google, DeepSeek, Ollama, NVIDIA, and more
BYOK
Bring your own keys. No lock-in
Local Models
Ollama support for fully offline use
MCP Support
Connect to MCP servers for custom tools
Session Branching
/fork conversations like Git
Sub-Agents
Spawn explorer, reviewer, debugger, builder
Permission Gate
Auto/confirm/plan/deny modes
Rich TUI
Ink-based terminal UI with cursor editing

Slash Commands

/model <p/m>    Switch model mid-chat
/use            Interactive model picker
/keys           Manage API keys
/fork           Branch conversation
/compact        Force context compaction
/resume         Session picker (↑↓/Enter/Esc)
/context        Token + context stats
/init           Guided setup wizard
/help           All commands
